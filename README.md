# Board Game Website

## Overview
This project is a board game website that showcases various board games, providing users with information, reviews, and the ability to contact the creators. The website is structured to be user-friendly and visually appealing, with a focus on accessibility and responsiveness.

## Features
- **Responsive Design**: The website is designed to be accessible on various devices, including desktops, tablets, and smartphones.
- **Game Listings**: A dedicated page to showcase available board games with detailed information.
- **Contact Form**: Users can reach out to the creators for inquiries or support.
- **About Page**: Information about the website and its mission.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd boardgame-website
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Open `src/index.html` in your web browser to view the website.

## Usage
- Navigate through the website using the header links.
- Explore different board games on the games page.
- Use the contact page to send messages to the creators.
